# Spotify-web-widget

Get the Spotify Web Widget.

[https://mollykannn.github.io/spotify-web-widget](https://mollykannn.github.io/spotify-web-widget)
![Screenshot](https://mollykannn.github.io/spotify-web-widget/img/screenshot.png)


## 安裝 (Install)

```shell
pnpm install
```

## 使用方法 (Usage)

建立檔案 (Create files)
```shell
pnpm run build
```

運行 (Run)
```shell
pnpm run serve
```
